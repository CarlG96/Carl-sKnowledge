# Directives are preceded by the # sign in C

# List of directives

- #define - Defines a macro

- #undef - Undefines a previous macro

- #include - includes a file

- #if - conditional compilation directive

- #ifdef - checks if a macro is defined

- #ifndef - checks if a macro is not defined

- #elif - Else if condition for #if

- #else - Else condition for #if

- #endif - Ends an #if, #ifdef, #ifndef block

- #error - Specifies diverse behaviour depending on compiler

- #pragma - Specifies diverse behaviour depending on compiler

- #line - Specifies the original line number and filename

- #warning - Generates a compiler warning with a specified message (GCC)
